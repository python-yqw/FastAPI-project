# FastAPI 学习项目
这是一个基于 Python FastAPI 开发的后端接口项目

项目包含：
- 基础接口示例
- 数据模型校验
- 自动生成 API 文档
- 可直接部署运行

## 一、环境准备
确保已安装 Python 3.8+  
下载地址：https://www.python.org/downloads/

##安装依赖
pip install -r requirements.txt

## 技术栈
-Python 3.8+
-FastAPI
-Uvicorn
-Pydantic
-Pandas & NumPy

## 启动命令
uvicorn main:app --reload
